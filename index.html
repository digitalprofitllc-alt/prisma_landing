¿Qué aprenderás?

Este curso está pensado para quienes trabajan con datos, independientemente de sus conocimientos técnicos o analíticos.

Aprenderás a:

  • Utilizar Dataiku, una de las plataformas más potentes para Inteligencia Artificial y Machine Learning de bajo código y de autoservicio.
  • Domina la investigación y estructura de proyectos con políticas de Gobierno AI.
  • Limpia y transforma datos de forma eficiente.
  • Construye flujos de trabajo de diferentes casos de uso.
  • Realiza análisis exploratorios, estadísticas descriptivas y predictivas.
  • Visualiza resultados con dashboards de analítica avanzada.

Una vez dominado el análisis descriptivo, el siguiente paso es avanzar hacia el análisis predictivo y prescriptivo. Con el uso de inteligencia artificial y machine learning, es posible incorporar recomendaciones generadas por algoritmos modelados, mejorando la toma de decisiones.
